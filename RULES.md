# Dragon Spin — Game Rules

## Basic Setup

- **Reels:** 5 reels, 3 visible rows each
- **Paylines:** 30 fixed paylines
- **Wins:** Left-to-right on active paylines, starting from reel 1. Only the highest win per line is paid.
- **Scatter:** Bonus symbol pays are independent of paylines (any position on reels 2, 3, 4). Scatter pay is multiplied by total bet.

## Symbols

### Regular Symbols
- **Dragons** (high value): Red, Blue, Purple, Green — all pay x50 / x20 / x5 for 5/4/3 of a kind
- **Cards** (low value): A pays x30/x10/x5. K, Q, J, 10 all pay x20/x10/x2 for 5/4/3 of a kind

### Special Symbols
- **Wild:** Substitutes for all symbols except Bonus. Only appears through Mystery Stacked Reels in the base game.
- **Bonus:** Scatter symbol. Appears on reels 2, 3, and 4 only during the base game. 3 Bonus symbols trigger the Dragon Spin Bonus Feature.

## Mystery Stacked Reels

Each reel has stacked "mystery" positions. When the spin starts:
- Each reel's mystery positions are all replaced with the same randomly chosen symbol
- The symbol is chosen from any available symbol **except Bonus**
- Wild can only appear through this mystery stack mechanic in the base game
- Each reel picks independently (1 of 2-3 possible symbols)

## Dragon Spin Bonus Feature

Triggered by landing 3 Bonus scatters on reels 2, 3, and 4. A wheel spins to select one of three free game modes:

### 1. Raining Wilds Free Games
- **Spins:** 5
- **Mechanic:** At the start of each spin, 3 to 10 Wild symbols are randomly placed on the reels and held in place while the remaining positions spin
- **Mystery stacks:** Still active, but resolve to any symbol except Wild
- **No retriggering:** Bonus symbols do not appear; no additional free spins awarded

### 2. Persisting Wilds Free Games
- **Spins:** 5
- **Mechanic:** Persisting Wild symbols appear randomly and lock into their position for the rest of the bonus round
  - Spins 1-3: may award 1-2 Persisting Wilds each
  - Spin 4: may award 1-3 Persisting Wilds
  - Spin 5: may award 2-7 Persisting Wilds
- **Mystery stacks:** Still active, resolve to any symbol except Persisting Wild
- **No retriggering:** Bonus and regular Wild do not appear; no additional free spins

### 3. Reel Blast Free Games
- **Spins:** 5
- **Mechanic:** Play occurs on 3 separate reel sets simultaneously, all sharing a common center reel (reels 2, 3, and 4)
  - The shared center reel shows one symbol that counts for all 3 reel sets
  - Reels 1 and 5 are independent per reel set
  - Each reel set uses the same 30 paylines
- **Mystery stacks:** Still active, all mystery positions on a reel resolve to the same symbol
- **No retriggering:** Bonus symbols do not appear; no additional free spins

## General Rules

- All pay amounts are multiplied by the line bet (except scatter, which uses total bet)
- Only the highest win per payline is paid
- Wild substitutes for everything except Bonus
- Each spin outcome is independent
- Theoretical RTP: 95.94%
